# Shortest Road Mapper (kliensoldali)

Megnyitás: nyisd meg az `index.html` fájlt közvetlenül böngészőben (nincs szükség szerverre).

1. Tölts fel egy képet (PNG/JPG/JPEG/BMP).
2. Kattints a bal oldali képen egy színre (automatikus „pipetta”).
3. Állítsd a **Threshold** csúszkát, a jobb oldali előnézet azonnal frissül.
4. Az **OK** gomb a szín kiválasztása után aktiválódik (lime).
5. Az aktív (szűrt) képen kattints két nem átlátszó pixelre: kezdőpont → végpont. A rendszer A* algoritmussal megkeresi a legrövidebb utat.

Gyorsbillentyű:
- `R`: pontok/útvonal törlése (Stage 2)
